# Algebra-Linear-Computacional
Pesquisa: Simulação Quântica de Inibidores NDM-1 (Superbactérias) via Ansatz Borromeano e Médias Metálicas (UNIFEI)

## Estrutura das aulas (v1)

Esta trilha organiza os estudos de Álgebra Linear Computacional em módulos progressivos, com foco em base matemática, métodos numéricos e aplicações.

### Formato padrão de aula

Cada aula deve conter:

- **Tema**
- **Objetivos de aprendizagem**
- **Pré-requisitos**
- **Duração sugerida**: 2h
- **Materiais de apoio**
- **Atividade prática**
- **Entregável**

### Periodicidade e duração do ciclo

- **Periodicidade**: semanal
- **Duração total**: 16 aulas (4 módulos + aulas de integração e revisão)

### Cronograma v1 (sequência pedagógica)

1. Espaços Vetoriais  
2. Bases e Dimensão  
3. Espaços com Produto Interno  
4. Transformações Lineares  
5. **Revisão Módulo 1 + Lista aplicada**
6. Autovalores e Autovetores  
7. Decomposição Espectral  
8. **Integração Módulo 2 (teoria + computação)**
9. Eliminação Gaussiana  
10. Decomposição LU  
11. Decomposição QR  
12. Decomposição SVD  
13. Condicionamento e estabilidade numérica  
14. **Integração Módulo 3 (estudo de caso computacional)**
15. Aplicações à Geometria Analítica + Tópicos em Análise Numérica  
16. Teoria dos Números + Criptografia Básica + fechamento

### Resultados de aprendizagem por módulo

- **Módulo 1 (Fundamentos)**: modelar problemas em espaços vetoriais, operar com bases, dimensão e transformações lineares.
- **Módulo 2 (Estruturas Espectrais)**: interpretar e aplicar autovalores/autovetores e decomposição espectral.
- **Módulo 3 (Métodos Computacionais)**: resolver sistemas e fatorações (Gauss, LU, QR, SVD) com análise de condicionamento.
- **Módulo 4 (Aplicações e Extensão)**: conectar álgebra linear com aplicações geométricas, numéricas e fundamentos de criptografia.
